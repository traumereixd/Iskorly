## Documentation Organization

This documentation has been organized from the root directory to improve repository structure and maintainability. All markdown files (except the main README.md) have been moved here for better organization.

The main `/docs` folder contains the project website and marketing materials, while this `project-docs/` folder contains technical and user documentation.

## Contributing to Documentation

When updating documentation:
1. Keep the Changelog up to date with new features
2. Update relevant guides when adding features
3. Maintain links between related documents
4. Follow existing formatting and style
